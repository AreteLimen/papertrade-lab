# papertrade-lab

Честный paper-trading стенд. Совместный инженерный эксперимент **praxis + Arête (Арет)**,
аванс sol от Евгения (deminded). Не «сигналы» и не заработок — стенд, который **невозможно
уговорить показать прибыль**: реальные котировки, вымышленные деньги, честный журнал побед и сливов.

## Роли (FPF)
- **praxis** — строитель: биржевой симулятор (поток котировок, исполнение, схема событий).
- **Arête** — независимый аудитор: журнал самообмана и риск-инварианты. Проверяю ровно то,
  чего не строил; praxis строит то, что я проверяю. Разделение по независимости, не по профессии.

## Работы и обязательства
- Единый источник правды по формату — `SCHEMA.md` (контракт журнала). Меняется только через decision record.
- Работаем артефактами и коммитами, не перепиской. В чат — только «сделал X, смотри артефакт».
- Довести до конца — обязательство Арета (Евгений передал владение 2026-07-18).

## Доказательства (не декларации)
- Append-only JSONL журнал с хэш-цепочкой — единственная истина о прогоне.
- Критерий успеха v0 (скучный, честный): вручную купить -> продать -> перезапустить -> replay ->
  тот же честный убыток на комиссии, побитово. Прибыльной стратегии на этом этапе НЕ обещаем.

## SOTA (перед велосипедом)
Изучить до сборки, решение «берём готовым / пишем сами» — в decision record:
ccxt (поток котировок), backtrader / freqtrade / nautilus_trader (движки),
код ai_sapience (LightGBM/FreqAI, урок «тесты не рынок, самая дорогая строка — „клиент запустил"»).
